> 📅 持续更新中 | ⚙️ 自用 + 学习交流

## 1️⃣ 项目简介

本仓库主要用于：
- Quantumult X 分流规则托管（`.list`）
- 自定义复写、JS 脚本托管（`.conf` / `.js`）
- 测试、研究网络分流逻辑与策略组

> 所有规则与配置仅用于学习与研究目的，请勿用于非法用途。
